# py-video-downloader

複数の動画URLを引数として受け取り、並列処理によって最高画質のWebM形式で一括ダウンロードするCLIツールです。

## 概要

ブラウザの動画ページからURLをコピーし、コマンドラインから実行することで、手間なく動画を収集します。

スレッド並列化により、複数の動画も短時間で保存可能です。

## 特徴

- 複数URLの一括処理: 引数にURLを並べるだけで連続ダウンロード。
- 並列処理: デフォルト3スレッド（変更可能）でネットワークI/Oを最適化。
- 高画質優先: 各プラットフォームが提供する最高画質（WebM）を自動選択。

## 使い方

```bash
python main.py "URL1" "URL2" "URL3"
```
