# py-video-downloader

複数の動画URLをコマンドライン引数から受け取り、並列処理によって最高画質の動画（WebM形式等）を一括ダウンロードするCLIツールです。

## 概要

`yt-dlp`ライブラリを利用し、YouTubeをはじめとする多数の動画プラットフォームから効率的に動画を収集します

。複数のURLを同時に処理できるようスレッド並列化が行われており、ネットワーク帯域を最大限に活用してダウンロード時間を短縮します。

## 主な機能

- **一括ダウンロード**: スペース区切りで複数のURLを指定し、順次または並列に処理。
- **並列実行のカスタマイズ**: `-p` オプションにより、同時ダウンロード数を自由に設定可能。
- **最高画質の自動選択**: 利用可能な最高画質の映像と音声を自動的に選択して保存。
- **SSLエラー対策**: macOS環境で発生しやすい証明書エラーを回避する設定を標準搭載。

## 環境構築

### 外部ツールのインストール

高画質な動画の結合処理などを行うため、システムに `ffmpeg` がインストールされている必要があります。

```bash
# macOSの場合
brew install ffmpeg
```

### Pythonライブラリのインストール

以下のコマンドで依存ライブラリをインストールしてください。

```bash
pip install yt-dlp
```

### macOSでのSSL証明書セットアップ

macOSで実行時にSSLエラーが発生する場合は、以下の手順を実行してください。

1. **/Applications/Python 3.x/** フォルダを開く。
2. **Install Certificates.command** を実行する。

## 使い方

```bash
# 単一の動画をダウンロード
python main.py "URL1"

# 複数の動画を3並列（デフォルト）でダウンロード
python main.py "URL1" "URL2" "URL3"

# 同時ダウンロード数を5に設定して実行
python main.py "URL1" "URL2" "URL3" "URL4" "URL5" -p 5
```

- **保存先**: デフォルトでは `~/Downloads/MyDownloadedVideos` フォルダが自動作成され、そこに保存されます。
